What's inside

Everything you need to understand matrix addition

Built for clarity first. Every feature is designed so that a student who has never seen matrix math before can follow along.

Matrix Addition

Add any two same-sized matrices from 1×1 up to 8×8. Values are validated on both the frontend and the server before any calculation runs.

Step-by-Step Solutions

After every calculation a full breakdown appears — one card per cell showing a + b = result, plus a row-by-row summary table and a highlighted result panel.

Dynamic Matrix Sizes

Change the row and column count on the fly. The input grids rebuild instantly and the step viewer always matches the exact dimensions you chose.

Calculation History

Every result is saved to the database under your account. Re-open any past calculation from the sidebar with one click — your data persists across sessions.

User Authentication

Secure sign-up and login with hashed passwords. Each user has a private history — no one else can see or delete your calculations.

Input Validation

Non-numeric inputs, mismatched dimensions, and oversized matrices are all caught with clear, friendly error messages — both in the browser and on the server.
